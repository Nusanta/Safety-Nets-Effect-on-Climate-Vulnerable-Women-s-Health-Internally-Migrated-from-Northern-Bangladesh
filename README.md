# Safety Nets Effect on Climate-Vulnerable Women’s Health  
### Internally Migrated from Northern Bangladesh

## Overview
This repository contains a mixed-methods analysis of climate-displaced women living in informal settlements in Dhaka (n = 120 households).

## Key Findings
- Safety nets improve employment transitions after migration  
- Safety nets do not independently improve health outcomes  
- Access to healthcare services is the strongest predictor of health  

## Insights
Findings highlight how structural barriers, including documentation, costs, and limited service access, shape vulnerability beyond income support mechanisms.

## Methods
- Household survey (n = 120; 342 individuals)  
- Focus group discussions (FGDs)  
- In-depth interviews (IDIs)  
- Quantitative analysis in R (logistic regression)  
- Qualitative thematic analysis  

## Repository Structure
- `/data` – Survey and processed datasets (if shareable)  
- `/scripts` – R scripts for data cleaning and analysis  
- `/outputs` – Tables, figures, and model results
